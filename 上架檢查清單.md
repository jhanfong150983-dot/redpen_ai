## 階段 3: 應用套件準備

### Android (Google Play)

#### PWABuilder 生成

- [ ] 使用 PWABuilder 生成 Android 套件
- [ ] 下載 `.zip` 檔案並解壓
- [ ] 取得 `app-release-signed.aab` 檔案
- [ ] 取得 SHA256 指紋

#### Digital Asset Links

- [ ] 建立 `public/.well-known/assetlinks.json`
- [ ] 填入正確的 Package ID: `com.redpenai.app`
- [ ] 填入正確的 SHA256 指紋
- [ ] 部署到 Vercel
- [ ] 驗證可訪問: `https://your-domain/.well-known/assetlinks.json`

#### AndroidManifest.xml 權限

- [ ] 相機權限: `android.permission.CAMERA`
- [ ] 網路權限: `android.permission.INTERNET`
- [ ] 儲存權限 (如需要)

#### 測試

- [ ] 在實體 Android 裝置安裝 APK
- [ ] 確認無網址列 (TWA 模式)
- [ ] 相機功能正常
- [ ] 所有功能可用

---

### iOS (App Store)

#### PWABuilder 生成

- [ ] 使用 PWABuilder 生成 iOS 套件
- [ ] 下載並在 Mac 上用 Xcode 開啟
- [ ] 配置 Bundle ID: `com.redpenai.app`

#### Info.plist 權限說明

- [ ] `NSCameraUsageDescription`: "RedPen AI 需要使用相機拍攝學生作業,以便進行 AI 批改並提供個人化學習建議。"
- [ ] `NSPhotoLibraryUsageDescription`: "允許從照片庫選擇作業圖片上傳。"
- [ ] `WKAppBoundDomains`: 您的 Vercel 網址

#### Capacitor 整合 (相機功能)

- [ ] 安裝 Capacitor: `npm install @capacitor/core @capacitor/ios @capacitor/camera`
- [ ] 修改 `CameraCapturePage.tsx` 加入 Capacitor fallback
- [ ] 測試相機功能

#### 測試

- [ ] 在實體 iPhone 測試 (模擬器無法測試相機)
- [ ] 確認相機權限對話框顯示正確
- [ ] 所有功能正常運作
- [ ] 沒有 WKWebView 錯誤
